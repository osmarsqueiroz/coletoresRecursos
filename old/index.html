Algoritmo A*(A estrela) - em busca do caminho menor

Este codigo, tenta buscar o melhor caminho no mapa desenhado

Objetivos

  1. Criar um mapa de quadros
  2. Criar o ponto inicial
  3. Criar o ponto final
  4. Criar barreiras

Extrar

  1. Fazer o ponto inicial buscar todos os pontos finais
  2. Fazer o caminho reverso para encontrar o melhor caminho
  3. Salvar mapas gerados
